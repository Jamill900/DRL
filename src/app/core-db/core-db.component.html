CORE DB

Making data available for everyone

Challenge

Core data plays an essential role in the decision-making process at different stages of the field development of any oil and gas field.

The customers problem was that the available large volume of core data is stored in an unstructured manner with no conventional naming practices scattered across the numerous folders of different technical personnel. Unavailability of an integrated database created additional effort and inefficiency of pre-conditioning, sorting and manual transfer of required data between different disciplines, which was diminishing the potential value that core data could bring in.

PROJECT OBJECTIVES:
QUALITY

QC and expand existing Core database.

UNIFORMITY

Establish core inventory structure and standards.

ACCESSIBILITY

Develop a web platform deployed on the customer’s servers making it readily available for all users.

VALUE

Boost the value of core data by removing inefficiencies and additional effort required for pre-conditioning, sorting.

Our Solution

At the pre-development stage, throughout close collaboration with the customer's technical and domain knowledge specialists, our team:

✓ Identified technical objectives and business needs.

✓ Established nomenclature and conventions for different type of core measurements.

✓ Set up the database architecture and business logics.

A fully automated web-based application with a user-friendly user interface was developed and deployed.
Alongside this, our team carried out an extensive QC and processing campaign of the existing data.

The application features:

✓ Interactive pages dedicated for each core measurement group

✓ Separate admin functionality that will enable subject matter data managers to maintain the database

✓ Integration with in house mapping platform (OneMap)

Business Benefits

✓ Readily accessible, well-sorted, fully standardized and validated core database from anywhere in the world.

✓ Integration of different core measurements.

✓ Potential to scale up to other regions and fields.
