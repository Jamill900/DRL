Core DB

Making data available for everyone

Core data plays an essential role in the decision-making process at different stages of the field development starting from the early exploration and appraisal phase through the entire extent of the development. The presence of readily available, sorted, QC-ed and unified core measurements can boost the efficiency during the key decision-making points such as reserve re-evaluation, new well delivery, lower completion selection as well as aid subject matter experts during the reservoir related studies and research.

The customers problem was that the available core data is stored in an unstructured manner with no conventional naming practices scattered across the numerous folders of different technical personnel. Hence, unavailability of an integrated database created additional effort and inefficiency of pre-conditioning, sorting and manual transfer of required data between different disciplines, which was diminishing the potential value that core data could bring in.

The purpose of the project was to come up with a well-sorted and fully standardized core database that provides users with multiple functionalities including but not limited to smart search, multi-filtering, visualization, access through commonly used industry software. A fully automated web-based application with a user-friendly user interface was developed and deployed on the customer's servers. The application comprises several pages dedicated for each data type as well as a separate admin functionality that will enable subject matter data managers to maintain the database. Alongside the development of the web application, our team processed and QC-ed all reports and partially QC source lab measurements, digitize pdf files where needed, combine and clean excel sheets, extract necessary images from lab reports and organize the database structure through close collaboration with the customer's core experts.
